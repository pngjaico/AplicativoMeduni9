Editor de JSON: Cole aqui o conteúdo de flashcards ou questões em formato JSON. O editor validará o formato automaticamente.

Como Adicionar Novos Códigos de Acesso

  1. No painel, clique em "+ Gerar Novo Código"
  2. Um novo código será criado automaticamente no formato XXXX-XXXX-XXXX
  3. Configure o tipo de plano (Premium, Standard, Basic) editando o arquivo data/codigos.json
  4. Defina o email associado (opcional) e a contagem de dispositivos
  5. Salve o arquivo para aplicar as mudanças

Como Adicionar Novos Flashcards

  1. Abra o arquivo data/flashcards.json
  2. Use a aba "Editor de Dados" para validar o JSON
  3. Estrutura esperada:
{
  "materias": [
    {
      "id": "anatomia",
      "nome": "Anatomia",
      "flashcards": [
        {
          "id": 1,
          "pergunta": "Qual é o maior osso do corpo?",
          "resposta": "O fêmur"
        }
      ]
    }
  ]
}

Como Adicionar Novas Questões

  1. Abra o arquivo data/questoes.json
  2. Estrutura esperada:
{
  "materias": [
    {
      "id": "anatomia",
      "nome": "Anatomia",
      "questoes": [
        {
          "id": 1,
          "pergunta": "Qual osso é esse?",
          "opcoes": ["A", "B", "C", "D"],
          "respostaCorreta": 0,
          "dificuldade": "média"
        }
      ]
    }
  ]
}

Deploy para Firebase

  1. Instale o Firebase CLI: npm install -g firebase-tools
  2. Faça login: firebase login
  3. Inicialize o projeto: firebase init
  4. Implante: firebase deploy

Atualizar a Aplicação

  1. Edite os arquivos JSON em data/
  2. Teste as mudanças localmente
  3. Faça commit: git add . && git commit -m "Update content"
  4. Envie para o repositório: git push
  5. Deploy automático via Firebase Hosting ou execute firebase deploy
